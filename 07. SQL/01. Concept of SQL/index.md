# **SQL**(Structed Qury Language)

## **SQL이란?**
- MySQL과 같은 **관계형 데이터베이스**에서 데이터를 읽거나 생성 및 수정하기 위해 생겨난 언어

- 기본적으로 **CRUD**라고 함.
    - **Create** (생성)
    - **Read** (읽기)
    - **Update** (수정)
    - **Delete** (삭제)

- 관계형 데이터베이스 시스템 **전용 언어**
