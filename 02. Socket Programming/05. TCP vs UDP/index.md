# **TCP vs UDP**

## 1. **공통점**

- **포트 번호**를 이용하여 주소를 지정한다.
    - 두 어플리케이션이 통신하기 위해서는 반드시 포트 번호를 결정해햐 한다.

- **데이터 오류**를 체크한다.
    - 헤더와 데이터에 대하 오류를 체크한다.

<br>

## 2. **차이점**

- **연결형 vs 비연결형**
    - TCP는 연결이 성공해야 통신이 가능하지만, UDP는 연결없이 통신이 가능하다.

- **바이트 경계 구분의 유무**
    - TCP: 바이트 스크림 (Byte-stream) 서비스
    - UDP: 데이터 그램 (datagram) 서비스

- **데이터의 신뢰성**
    - TCP: 데이터를 재전송함.
    - UDP: 데이터를 재전송 하지 않음.

- **통신**
    - TCP
        - 1 대 1 통신(unicast)
        
    - UDP
        - 1 대 1 통신(unicast)
        - 1 대 다 통신(broadcast)
        - 1 대 다 통신(multicast)
